## MH4110: Partial Differential Equations – Presentation Materials

This repository contains the **LaTeX** source code and project assets for a technical presentation based on the paper: **"A Neural Network Approach to Solve the Black-Scholes Equation"** ([arXiv:2405.05780](https://arxiv.org/abs/2405.05780)).

The project was completed as part of the **MH4110 (Partial Differential Equations)** course requirements.
